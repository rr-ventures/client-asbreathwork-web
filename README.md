# ASB Breathwork Web Client

A professional website for Altered States Experience, a breathwork instruction business.

## Client Information

**Business Name:** Altered States Experience  
**Contact:** gene@gkfitness.com.au  
**Industry:** Breathwork Instruction

## Project Goals

### Primary Objectives
- Legitimize the business and present as experienced professionals
- Build trust and credibility to attract clients
- Create a professional online presence that encourages business engagement

### Key Challenges
- **Current Strength:** Excellent workshop system
- **Current Challenge:** Difficulty selling higher-ticket offers
- **Solution Strategy:** 
  - Face-to-camera social media campaign to build know, like, and trust
  - Professional website to establish credibility and showcase expertise

### Website Requirements
- Slick, eye-catching, and clean design
- Accurately represents the breathwork services offered
- Lead capture functionality (newsletter sign-ups)
- Showcase upcoming events and workshops
- Help visitors get to know the instructors
- Build trust and credibility

### Design Reference
The client likes the design and style of: https://www.zenthaishiatsu.com

## Development Setup

This project uses a Dev Container for consistent development environments.

### Getting Started

1. Open the project in VS Code
2. Press `F1` (or `Ctrl+Shift+P`)
3. Select "Dev Containers: Reopen in Container"
4. The container will build and install dependencies automatically

### Project Structure

```
client-asbreathwork-web/
├── .devcontainer/     # Dev container configuration
├── .gitignore        # Git ignore rules
└── README.md         # This file
```
